# BGTraffic.eu Data Collector v2.1.0

Събира и публикува:

- `latest.json` — текущ трафик от БГТОЛ;
- `weather.json` — пътна метеорология от официалната страница за отворени данни на БГТОЛ;
- `accidents.json` — ПТП чрез ArcGIS REST конфигурацията на официалната карта на МВР.

Трафикът се публикува преди допълнителните колектори, така че грешка във времето или ПТП да не оставя live feed-а остарял.
